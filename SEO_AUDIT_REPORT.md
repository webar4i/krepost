## ⚠️ КРИТИЧЕСКИЕ ПРОБЛЕМЫ

### 1. Title слишком длинный
**Проблема:** Title на главной = 69 символов (рекомендуется 50-60)
```html
Сейчас: "Строительство домов под ключ в Махачкале | Проект, инженерия, отделка"
```
**Решение:**
```html
"Строительство домов в Махачкале под ключ | КРЕПОСТЬ"
```

### 2. Отсутствуют важные Schema.org типы
**Проблема:**
- ❌ Нет WebPage Schema
- ❌ Нет BreadcrumbList Schema
- ❌ Нет Organization Schema

**Решение:** Добавить в `<head>`:
```json
{
  "@context": "https://schema.org",
  "@type": "WebPage",
  "name": "Строительство домов в Махачкале",
  "url": "https://krepostdom.ru/",
  "breadcrumb": "Главная"
}
```

### 3. Внешние изображения
**Проблема:** Все 8 изображений загружаются с Unsplash (внешний домен)
- ❌ Нет контроля над изображениями
- ❌ Зависимость от третьего лица
- ❌ Нет WebP формата

**Решение:**
1. Скачать все изображения локально
2. Конвертировать в WebP
3. Добавить lazy loading

### 4. Нет минификации
**Проблема:** 0 минифицированных файлов из 9 (CSS + JS)

**Решение:**
```bash
# Установить инструменты
npm install -g terser clean-css-cli

# Минифицировать
cleancss -o main.min.css main.css
terser -o main.min.js main.js
```
