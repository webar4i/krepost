# First 14 Days Monitoring Plan

## Daily
- Проверка 404/5xx и индексации ключевых URL.
- Проверка лидов и событий (form_submit/call_click).
- Проверка заметных просадок трафика (>20% day-to-day).

## Weekly
- Позиции по приоритетным кластерам.
- CR и micro-conversions по шаблонам страниц.
- QA новых страниц (meta, schema, internal links).

## Day 14 checkpoint
- Сравнение факта с baseline по органике и лидам.
- Список корректирующих задач на спринт 2.
