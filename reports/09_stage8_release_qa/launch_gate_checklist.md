# Launch Gate Checklist

## GO only if
- Все ключевые URL отдают `200`.
- Нет критичных ошибок schema/metadata.
- Формы и клики по контактам отправляют события в аналитику.
- Proof Matrix закрыта по всем обещаниям Hero/CTA.
- Публикация в sitemap и отправка в вебмастера выполнены.

## STOP conditions
- Нет рабочих форм на mobile.
- Важные страницы закрыты robots/noindex.
- Нет подтверждения по юридически значимым обещаниям (цены/гарантии).
