# Page Templates (адаптация из `seo.md`)

## 1) SERVICE LANDING
- Цель: транзакционный интент (лид/звонок)
- Блоки: Hero -> Цена/диапазоны -> Что входит -> Этапы -> Кейсы -> Отзывы -> FAQ -> CTA
- Объем: 900-1800 слов

## 2) GUIDE PILLAR
- Цель: информационный интент + ассист в коммерческую страницу
- Блоки: Введение -> Короткий ответ -> Критерии -> Процесс -> Ошибки -> FAQ -> Чек-лист -> CTA
- Объем: 2500-4000 слов

## 3) PRICES PAGE
- Цель: снять ценовое возражение
- Блоки: Короткий ответ -> Таблица цен -> Факторы стоимости -> Примеры расчетов -> FAQ -> CTA
- Объем: 700-1500 слов

## 4) REVIEWS/CASES HUB
- Цель: E-E-A-T и social proof
- Блоки: внешние площадки -> лента отзывов/кейсов -> ответы компании -> CTA
