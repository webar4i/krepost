# Dashboard Spec

## Вкладки
1. SEO performance
2. Leads & conversion
3. SERP & competitors
4. AEO/LLM monitoring
5. Local/Off-page

## Основные виджеты
- Organic traffic trend (week/month)
- Leads from organic (form/call/messengers)
- Top queries by intent and position bucket
- Cluster coverage progress (published pages vs plan)
- AEO mentions/citations log
- Local profile actions (calls, route, clicks)

## Алерты
- Падение органики >20% WoW
- Ошибки индексации >5 новых URL
- Просадка conversion rate >15% WoW
