## 🔴 КРИТИЧНО: Действия, которые нужно выполнить

### 1. Активация GitHub Pages

1. Зайдите в настройки репозитория на GitHub
2. Перейдите в раздел **Settings → Pages**
3. В разделе "Source" выберите:
   - Branch: `main`
   - Folder: `/ (root)`
4. Нажмите **Save**
5. Дождитесь публикации (обычно 1-5 минут)
6. Сайт будет доступен по адресу: `https://webar4i.github.io/krepost_doma/`

### 2. Настройка аналитики

Отредактируйте файл `assets/js/analytics-config.js`:

```javascript
window.ANALYTICS_CONFIG = {
  // Google Analytics 4 Measurement ID (формат: G-XXXXXXXXXX)
  googleAnalyticsId: 'G-XXXXXXXXXX',  // ← ВСТАВИТЬ РЕАЛЬНЫЙ ID
  
  // Яндекс.Метрика counter ID (число)
  yandexMetrikaId: 123456789,  // ← ВСТАВИТЬ РЕАЛЬНЫЙ ID
  
  debug: false
};
```

### 3. Заполнение реальных данных компании

## 🔧 Отправка в поисковые системы

### Яндекс.Вебмастер
1. Перейдите на https://webmaster.yandex.ru/
2. Добавьте сайт `krepostdom.ru`
3. Подтвердите владение (через DNS или файл)
4. Отправьте Sitemap: `https://krepostdom.ru/sitemap.xml`

### Google Search Console
1. Перейдите на https://search.google.com/search-console
2. Добавьте свойство `krepostdom.ru`
3. Подтвердите владение
4. Отправьте Sitemap
